# 🚍 Bus Route Registration & Student Tracking System

## 🌟 Technologies Used
- **Backend**: Java (Servlets, JSP, JDBC)
- **Frontend**: HTML, CSS, JavaScript, Bootstrap
- **Database**: SQLserver
- **Unit Testing**: JUnit
- **Build Tool**: Apache Netbean 17, JDK17
- **Version Control**: Git & GitHub

- Real-time student tracking

- Vehicle issue reporting for drivers
- System management for admin

## 🚀 How to Run the Project
1. Clone this repository:
   ```bash
   [git clone https://github.com/your-repo.git](https://github.com/hoangvanduc2k4/EduBus.git)
   ```
2. Import project into your IDE (Eclipse/IntelliJ)
3. Set up database using `BUS_MANAGEMENT.sql` and `trigger.sql`
4. Configure `DBContext.java` with your database credentials
5. Deploy on Apache Tomcat Server
6. Open the browser and access: `http://localhost:9999/`

---
📌 **Note**: Ensure you have Java JDK, SSMS SqlServer, and Tomcat installed!
